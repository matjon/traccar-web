  4. YUI library (The Yahoo! User Interface Library)

    (Used by ExtJS)

    Original notice in the ./LICENSE file in ext-6.2.0-gpl.zip:

    Library: YUI 0.6 - drag-and-drop code
License: BSD
Location: http://developer.yahoo.com/yui

    According to notices in source code, several classes are based on code copied from the YUI library, most of which are concerned with drag-and-drop functionality.

    Firstly, the following classes are based on YUI:

    • Ext.dd.DragDropManager (defined in DragDropManager.js)
    • Ext.dd.DragDrop (defined in DragDrop.js)
    • Ext.dd.DD (defined in DD.js)
    • Ext.dd.DDProxy (defined in DDProxy.js)
    • Ext.dd.DDTarget (defined in DDTarget.js)

    These classes can be found in the ./classic/classic/src/dd/ directory in ExtJS sources and have the following notice in their header:

    /*
 * This is a derivative of the similarly named class in the YUI Library.
 * The original license:
 * Copyright (c) 2006, Yahoo! Inc. All rights reserved.
 * Code licensed under the BSD License:
 * http://developer.yahoo.net/yui/license.txt
 */

    Secondly, there is a notice in class Ext.env.Ready (defined in packages/core/src/env/Ready.js) that some code has been copied from YUI: 

    licensed courtesy of http://developer.yahoo.com/yui/license.html

    Thirdly, YUI is mentioned also in class Ext.event.Event (file classic/classic/overrides/event/Event.js). Maybe something was copied there, maybe not.

    Additionally, a notice in the file examples/classic/app/nested-loading/resources/json/products.json (search for "YUI" there) suggest that early versions of the ExtJS library may have been just extended versions of YUI. So maybe some more ideas were copied from YUI.

    What's more, there are several additional mentions of YUI TODO finish this

    The ancient version of the YUI library is (fortunately) not necessary for compiling ExtJS.

    Homepage: https://yuilibrary.com/

    Old homepage: http://developer.yahoo.com/yui
